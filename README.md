## ▶️ How to Use

1. Clone the repository:

```bash
git clone https://github.com/Mohamed-Irreef/Artificial_Intelligence__course.git
```

2. Open a topic folder (example: Python basics):

```bash
cd "Artificial_Intelligence__course/1. python_basics"
```

3. Run any script:

```bash
python datatypes.py
```

> Tip: If you don’t have packages installed yet, you can install them with:

```bash
pip install numpy pandas matplotlib seaborn
```
